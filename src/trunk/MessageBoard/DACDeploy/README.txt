This is a Data tier Application Framework (DAC) 
Console App for automating Database Provisioning
================================================
 
Execution Instructions
------------------

Prior to running this program, a dac.config file must be created.
An example dac.config file with the required format could be found in the project directory.
It contains settings that serves as argument or parameters to the program.

Settings includes:
 Source Database Name from which a bacpac export could be done  
 SqlAzure database account parameters 
 e.t.c.

 Modifying the format or not specifying the correct values may cause gotchas in program execution.
